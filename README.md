Example of use
--------------

```sh
./check_domain_expiration.sh -d google.com -w 60 -c 30
or
./check_domain_expiration.sh -d youdomain.com.au -s crazydomains.com
```

Requirements
------------

Plugin requires the following packages installed on your Icinga/Nagios node:

* WHOIS
* AWK
